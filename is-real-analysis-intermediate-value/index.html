Intermediate Value Theorem

Continuous functions on intervals attain every intermediate value

Concept Snapshot

The Intermediate Value Theorem (IVT) states that if f is continuous on [a,b] and k lies between f(a) and f(b), then there exists c ‚àà (a,b) such that f(c) = k. This intuitive result‚Äîcontinuous functions "hit" every value between their endpoints‚Äîformalizes the idea that continuous functions have no gaps. It guarantees the existence of roots, fixed points, and solutions to equations, making it fundamental to both theoretical and applied mathematics.

Bisection Method

Step 1 ‚Äì Assume f(a) < k < f(b). WLOG assume f(a) < k < f(b). Define sets A = {x ‚àà [a,b] : f(x) < k} and B = {x ‚àà [a,b] : f(x) ‚â• k}.
Step 2 ‚Äì Completeness. A is nonempty (contains a) and bounded above (by b), so sup A exists. Let c = sup A.
Step 3 ‚Äì Show f(c) = k. Suppose f(c) < k. By continuity, f(x) < k for all x in some neighborhood of c, so c is not the supremum‚Äîcontradiction. Similarly, f(c) > k leads to contradiction. Thus f(c) = k.

Alternative: Bisection Algorithm

Constructive approach. Repeatedly bisect [a,b]. At each step, choose the half where f switches from below k to above k (or vice versa). Nested intervals converge to a point c where f(c) = k by continuity and completeness.

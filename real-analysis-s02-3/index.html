Real Analysis: Differentiation & Integration

Part 3 · Riemann Integration

The Riemann integral provides a rigorous definition of area under a curve, extending geometric intuition to arbitrary bounded functions. Given a function f: [a, b] → ℝ, the integral ∫ₐᵇ f(x) dx is constructed by partitioning the interval, forming approximating sums, and taking limits as partition mesh tends to zero. A function is Riemann integrable if upper and lower sums converge to the same value. Continuous functions and monotone functions are Riemann integrable, as are functions with finitely many discontinuities. Integration provides tools for computing areas, volumes, arc lengths, and physical quantities like work and probability.

A partition P of [a, b] is a finite sequence a = x₀ < x₁ < ⋯ < xₙ = b dividing the interval into subintervals [xᵢ₋₁, xᵢ]. The mesh or norm of P, denoted ‖P‖, is the maximum subinterval length max(xᵢ − xᵢ₋₁). For a bounded function f, define the upper sum U(f, P) = Σ Mᵢ(xᵢ − xᵢ₋₁), where Mᵢ = sup{f(x) : x ∈ [xᵢ₋₁, xᵢ]}, and the lower sum L(f, P) = Σ mᵢ(xᵢ − xᵢ₋₁), where mᵢ = inf{f(x) : x ∈ [xᵢ₋₁, xᵢ]}. Upper sums overestimate area; lower sums underestimate.

Refinement improves approximation. If P' refines P (contains all points of P plus possibly more), then L(f, P) ≤ L(f, P') ≤ U(f, P') ≤ U(f, P). Taking suprema and infima over all partitions yields the lower integral inf U(f, P) and upper integral sup L(f, P). A function f is Riemann integrable on [a, b] if the lower and upper integrals coincide; their common value is ∫ₐᵇ f(x) dx. Equivalently, for every ε > 0, there exists a partition P with U(f, P) − L(f, P) < ε.

Continuous functions on closed bounded intervals are Riemann integrable. The proof uses uniform continuity: given ε > 0, choose δ such that |f(x) − f(y)| < ε/(b − a) whenever |x − y| < δ. Partition [a, b] with mesh less than δ; then Mᵢ − mᵢ < ε/(b − a), so U(f, P) − L(f, P) < ε. This argument demonstrates how compactness and continuity interact to guarantee integrability. Monotone functions are also integrable, as oscillation within subintervals can be controlled uniformly.

Riemann sums generalize upper and lower sums by choosing sample points freely. For a partition P and sample points cᵢ ∈ [xᵢ₋₁, xᵢ], the Riemann sum is R(f, P, {cᵢ}) = Σ f(cᵢ)(xᵢ − xᵢ₋₁). If f is integrable, then for every ε > 0, there exists δ > 0 such that |R(f, P, {cᵢ}) − ∫ₐᵇ f| < ε whenever ‖P‖ < δ, regardless of sample point choice. This criterion justifies numerical integration methods like midpoint, trapezoidal, and Simpson's rules, which approximate integrals via Riemann sums with specific sample strategies.

The integral satisfies linearity and order properties. If f and g are integrable on [a, b] and c ∈ ℝ, then ∫(f + g) = ∫f + ∫g and ∫(cf) = c∫f. If f ≤ g on [a, b], then ∫f ≤ ∫g. The absolute value inequality |∫f| ≤ ∫|f| follows from order properties. Additivity over intervals holds: ∫ₐᶜ f + ∫ᶜᵇ f = ∫ₐᵇ f for a < c < b. These properties enable algebraic manipulation of integrals and underpin comparison theorems and mean value inequalities.

Not all bounded functions are Riemann integrable. The Dirichlet function, defined as f(x) = 1 if x is rational and f(x) = 0 if x is irrational, is bounded on [0, 1] but not Riemann integrable: upper sums equal 1 while lower sums equal 0 for any partition. This example motivates the Lebesgue integral, which integrates a broader class of functions by partitioning the range rather than the domain. However, for continuous or piecewise continuous functions, Riemann integration suffices and provides geometric intuition.

Riemann integration transforms calculus from differentiation-focused to a balanced framework incorporating both local and global analysis. The Fundamental Theorem of Calculus, explored in the next article, reveals the deep connection between derivatives and integrals, showing that integration reverses differentiation. Understanding Riemann sums and integrability criteria equips the analyst with tools for computing areas, solving differential equations, and modeling accumulation processes across science and engineering.

Real Analysis: Advanced Topics

Part 4 · Metric Spaces

Metric spaces generalize the notion of distance from Euclidean space to arbitrary sets, providing a unified framework for convergence, continuity, and compactness. A metric space (X, d) consists of a set X and a distance function d: X × X → [0, ∞) satisfying non-negativity (d(x, y) ≥ 0 with equality if and only if x = y), symmetry (d(x, y) = d(y, x)), and the triangle inequality (d(x, z) ≤ d(x, y) + d(y, z)). Examples include Euclidean space ℝⁿ with d(x, y) = |x − y|, function spaces with supremum or integral norms, and discrete spaces with d(x, y) = 1 for x ≠ y.

The metric induces a topology via open balls. The open ball B(x, r) = {y ∈ X : d(x, y) < r} consists of all points within distance r of x. A set U ⊆ X is open if for every x ∈ U, there exists r > 0 such that B(x, r) ⊆ U. A set is closed if its complement is open. These definitions generalize the open and closed sets of ℝ, enabling topological reasoning in abstract spaces. For example, in the discrete metric, every set is both open and closed, while in ℝ with the usual metric, open intervals are open and closed intervals are closed.

Convergence in metric spaces uses the metric directly. A sequence (xₙ) converges to x if for every ε > 0, there exists N such that d(xₙ, x) < ε for all n ≥ N. Equivalently, xₙ → x if d(xₙ, x) → 0 in ℝ. Limits are unique in metric spaces due to the triangle inequality: if xₙ → x and xₙ → y, then d(x, y) ≤ d(x, xₙ) + d(xₙ, y) → 0, so x = y. This uniqueness, absent in general topological spaces, simplifies analysis and ensures determinism in limiting processes.

Continuity generalizes via metric preservation. A function f: (X, dₓ) → (Y, dᵧ) is continuous at x if for every ε > 0, there exists δ > 0 such that dₓ(x', x) < δ implies dᵧ(f(x'), f(x)) < ε. Equivalently, f is continuous at x if xₙ → x implies f(xₙ) → f(x). A function is continuous if it is continuous at every point. This definition encompasses continuity of functions between ℝⁿ spaces, uniform spaces, and function spaces, unifying diverse notions under a single framework. For instance, linear operators between normed spaces are continuous if and only if they are bounded.

Completeness in metric spaces mirrors completeness of ℝ. A sequence (xₙ) in X is Cauchy if for every ε > 0, there exists N such that d(xₘ, xₙ) < ε for all m, n ≥ N. A metric space is complete if every Cauchy sequence converges. Euclidean space ℝⁿ is complete, as are closed subsets thereof. The rationals ℚ with the usual metric are not complete: Cauchy sequences converging to irrationals do not converge in ℚ. Completeness is crucial for fixed-point theorems, contraction mapping principles, and existence proofs in differential equations.

Compactness extends the Heine-Borel characterization. A subset K of a metric space is compact if every open cover has a finite subcover. In metric spaces, compactness is equivalent to sequential compactness (every sequence has a convergent subsequence) and to being complete and totally bounded (for every ε > 0, K can be covered by finitely many ε-balls). In ℝⁿ, Heine-Borel identifies compact sets as closed and bounded. Compact metric spaces inherit many properties of finite sets: continuous functions attain extrema, and continuous bijections are homeomorphisms.

Examples of metric spaces illustrate the breadth of the theory. The space C([a, b]) of continuous functions with supremum norm ‖f‖_∞ = sup |f(x)| is a complete metric space, foundational in functional analysis. The space L²([a, b]) of square-integrable functions with d(f, g) = (∫ |f − g|²)^(1/2) is complete (a Hilbert space) and central to quantum mechanics and signal processing. Discrete metrics turn any set into a metric space, useful in combinatorics and computer science. These diverse examples demonstrate the unifying power of metric space abstraction.

Metric spaces provide the natural setting for generalizing real analysis to abstract contexts. Convergence, continuity, compactness, and completeness—concepts first developed for ℝ—extend seamlessly to metric spaces, enabling analysis of function spaces, sequence spaces, and geometric objects. Subsequent study leads to Banach spaces (complete normed vector spaces), Hilbert spaces (complete inner product spaces), and ultimately to general topology and manifold theory. Understanding metric spaces is essential for modern analysis, geometry, and applications across mathematics and physics.
